## CONWAY'S GAME OF LIFE (REACT + TYPESCRIPT)
node & npm versions

    :~ node -v
    v10.19.0
    :~ npm -v
    6.14.4

*Create a react app named 'go-life'*

    :~ npx create-react-app --template typescript go-life

*Navigate to folder 'go-life'*

    :~ cd go-life

*starting local server :3000*

    :~ npm start
